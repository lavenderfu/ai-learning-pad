# 产品需求变更记录 (Changelog)

## [v1.1] - 2026-03-13
**变更提出人**：PM / 核心用户
**变更类型**：中度变更（新增功能模块）

### 1. 新增功能
- **伴学吉祥物机制**：引入「卡通小鸡」作为伴学吉祥物。
- **动态欢迎/告别语**：
  - **欢迎逻辑**：基于浏览器的 LocalStorage 判断最后访问时间。如果连续访问，小鸡会给予日常夸赞；如果中断了天数，小鸡会说类似“xx你昨天怎么没有来看我，我自己真不好玩”的傲娇/思念语。
  - **告别逻辑**：学习任务完成后，小鸡会跳出并给予告别与期盼语（例如：“今天的新知识学完啦，你可真棒，明天别忘了还来找我玩哦”）。
  - 文案可由 AI 根据天数动态生成，或使用前端预置模板库。

### 2. 影响范围
- **用户流程**：在进入答题主界面前增加一个「欢迎全屏提示」，答题结束后替换/增加原有的星星通关动画为「小鸡告别全屏提示」。
- **前端开发**：需要增加小鸡动画组件，以及利用 `localStorage` 计算时间差的逻辑。
- **系统架构**：纯前端新增逻辑，无后端重大变更。
