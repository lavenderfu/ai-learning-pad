# 产品规格文档 (Product Spec)

## 1. 产品概述
### 1.1 产品定位
为3岁学前儿童定制的专注“数学逻辑”与“汉字启蒙”的 AI 互动学习平板 H5 应用。主打“互动即学习，零误触沉浸，AI 自动生成与自适应复习”。参考《数字积木》的具象化认知与《洪恩识字》的字形解构模式，确保孩子记住知识而非游戏本身。

### 1.2 目标用户
*   **直接使用者**：3岁儿童。特点：不识字、精细动作发育中、注意力集中时间短。
*   **付费与管理者**：家长。特点：希望孩子碎片时间有意义，但无暇每天手动配题配置，需要了解学习进度。

### 1.3 核心功能清单
1.  **AI 双擎任务系统**：支持家长手动配置/云端远程配置主题（AI 自动生成题库）+ AI 自适应每日复习推荐。
2.  **云端远程配置台**：家长在异地（如上班时）可通过云端配置文件设定今日学习主题，H5 学习端启动时自动拉取。
3.  **防误触沉浸学习界面**：全屏无菜单设计，仅留家长锁（长按 + 复杂算术题验证）。
4.  **听音点按互动 (Type A)**：AI 播报语音指令，大尺寸卡片点按选择，正向动效反馈。
5.  **具象化拖拽互动 (Type B)**：支持简单的方块/汉字偏旁拖拽组合，强化形体记忆。
6.  **家长数据看板**：可视化展示昨日学习报告与今日 AI 推荐路径。
7.  **伴学吉祥物机制（新增）**：包含开场欢迎与结束告别机制，基于本地存储判断连续访问天数。通过「卡通小鸡」形象结合语音进行正向情感反馈。

## 2. 界面与流程设计

### 2.1 用户核心流程
*   **路径1（孩子独立学习/异地模式）**：打开平板 H5 -> 自动读取任务 -> **显示卡通小鸡欢迎动画与问候语（基于最后访问天数判断逻辑）** -> 进入全屏沉浸学习 -> 完成后播放**小鸡告别动画与鼓励语** -> 等待家长解锁。
*   **路径2（家长本地配置）**：打开平板 H5 -> 进入家长台（输入解锁密码）-> 查看右侧 AI 推荐或手动输入今日主题（如“认识数字3”） -> 点击开始 -> 把平板给孩子。

### 2.2 整体布局规范 (UI)
*   **目标设备**：平板横屏 H5 优先。
*   **视觉风格**：现代极简 + 大色块，高对比度，保护视力，减少不必要的背景装饰。
*   **家长台（左右两栏）**：
    *   `左栏 (40%)`：数据看板。显示“昨日正确率”、“今日 AI 推荐主题”。
    *   `右栏 (60%)`：任务控制区。输入框“输入今日想学的主题（如：认识苹果/数字1-5）”，大大的“开始学习”按钮，以及“同步云端配置”状态。
*   **孩子学习台（全屏沉浸）**：
    *   `主区域 (100%)`：核心游戏互动区（卡片选择、拖拽区）。
    *   `吉祥物模块`：动画形象「卡通小鸡」，在学习开始和结束时占据屏幕主体进行播报。
    *   `角落部件`：半透明的小锁图标，长按 3 秒弹出家长验证框（如：3 × 4 = ?）。没有任何其他退出/返回按钮。

## 3. AI 能力需求
本产品极度依赖 AI 能力来实现家长减负与个性化学习。

1.  **文本生成大模型 (LLM)**：
    *   **用途 1：动态题库生成**。根据家长输入的模糊主题（如“数字积木2”）或云端配置文本，自动生成符合 3 岁认知逻辑的 JSON 格式题库数据（包含题目文案、正确选项、干扰项）。
    *   **用途 2：自适应学习策略**。根据前一天的答题日志（错题、耗时），生成第二天的复习和进阶主题。
    *   **用途 3：动态生成情感陪伴语（新增）**。根据孩子的连续打卡天数或中断天数，生成符合卡通小鸡人设的趣味打招呼/告别语。
2.  **语音合成 (TTS)**：
    *   **用途**：动态生成题库对应的语音播报（如：“小朋友，请把木字旁拖到右边”），语速需调慢，音色需亲切童声。

## 4. 技术与架构约束
*   **前端框架**：React/Vue + 拖拽库 (如 dnd-kit 或 PIXI.js 应对更复杂交互) 以实现流畅的平板 H5 体验。
*   **后端需求**：需要极简后端。
    *   提供云端配置文件（Config）的拉取/更新接口。
    *   提供学习日志记录接口。
    *   提供对接 LLM 接口的代理（Proxy），将生成的题目格式化为前端可读结构。
*   **数据存储**：由于涉及云端远程配置，家长指令需落库（可使用轻量级数据库或 BaaS 服务，如 Supabase/云开发）。

---

## 5. 验收标准
*   **第一期交付**：前端 H5 包含【家长看板】和【防误触答题台】两个视图；
*   能通过 JSON 模拟拉取云端任务；
*   能完成「听音点按」和「简单拖拽」两类核心题型的完整闭环交互，防退出锁逻辑生效。
