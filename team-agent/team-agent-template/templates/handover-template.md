## 如何运行项目

### Web 应用

```bash
# 进入项目目录
cd 2-build

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入实际值

# 启动开发环境
npm run dev

# 访问地址
http://localhost:{端口号}
```

### 微信小程序

```
1. 使用微信开发者工具打开 2-build/miniprogram 目录
2. 在云开发控制台配置环境变量（如有）
3. 上传并部署云函数（如使用云开发）
4. 点击「预览」或「真机调试」
```
